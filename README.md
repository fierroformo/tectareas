tectareas
=========
[![Build Status]development]

## Requirimientos

* Vagrant (http://www.vagrantup.com/)
* fabric (http://fabfile.org/)

## Configuraciones

1. Agregar `192.168.33.10   tectareas.local` a  /etc/hosts
2. Reiniciar apache `/etc/init.d/apache2 restart`

## Setup

1. `vagrant up`
